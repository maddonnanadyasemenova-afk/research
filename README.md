# Marketing Research AI Telegram Bot

Telegram-бот для проведения профессиональных маркетинговых исследований с помощью ИИ.

## Возможности

- 🎯 7 типов маркетинговых исследований
- 💰 Система баллов и оплата через Telegram Stars
- 📊 История всех исследований
- 🎁 Бонус 500₽ при первом запуске
- 🤖 Интеграция с Manus LLM API

## Типы исследований

1. **Определите свою целевую аудиторию** — 2000₽
2. **Генератор маркетинговой стратегии** — 4000₽
3. **Анализ каналов сбыта продукции/услуг** — 1500₽
4. **Определите проблемы вашей целевой аудитории** — 500₽
5. **Исследование международного рынка** — 700₽
6. **Карта пути клиента** — 1500₽
7. **Разработать программу лояльности клиентов** — 700₽

## Установка

```bash
# Установка зависимостей
pip3 install -r requirements.txt

# Настройка переменных окружения
# Отредактируйте .env файл с вашими данными
```

## Запуск

```bash
python3 bot.py
```

## Структура проекта

- `bot.py` - Основной файл бота с обработчиками
- `database.py` - Работа с SQLite базой данных
- `llm_client.py` - Клиент для Manus LLM API
- `researches.json` - Конфигурация типов исследований
- `.env` - Переменные окружения (токены, ключи)

## Команды бота

- `/start` - Начать работу с ботом
- `/research` - Начать новое исследование
- `/history` - История исследований
- `/balance` - Проверить баланс
- `/topup` - Пополнить баланс

## База данных

Бот использует SQLite с тремя таблицами:

- `users` - Пользователи и их баланс
- `researches` - История исследований
- `transactions` - История транзакций

## Оплата

Оплата реализована через Telegram Stars:
- 1 Star = 10 рублей
- Доступные пакеты: 500₽, 1000₽, 2000₽, 5000₽

## Примечания

- Бот работает на базе Manus LLM API (не требует отдельного OpenAI ключа)
- Все результаты исследований отправляются прямо в Telegram
- История исследований сохраняется и доступна в любое время
